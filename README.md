# Digital Entertainment Consumption Analysis

## 📊 Project Overview
This project analyzes how digital entertainment consumption affects academic performance and lifestyle balance among university students.

## 🎯 Objectives
- Understand screen time patterns
- Measure Entertainment Addiction Score (EAS)
- Analyze impact on Academic Performance Index (API)
- Identify behavioral clusters using PCA & K-Means

## 🧠 Methods Used
- Exploratory Data Analysis (EDA)
- Correlation Analysis
- Multiple Linear Regression
- PCA (Principal Component Analysis)
- K-Means Clustering
- ANOVA

## 🛠 Tools & Technologies
- R (tidyverse, ggplot2, factoextra)
- Quarto

## 📁 Project Structure
- scripts/ → analysis code (.qmd)
- output/ → HTML dashboard/report
- report/ → final academic report

## 📈 Key Insights
- Higher entertainment addiction → lower academic performance
- Screen time > 4 hours reduces consistency
- High-risk users cluster in heavy screen usage

## 🚀 How to Run
1. Open `.qmd` file in RStudio
2. Install required packages
3. Run / Render using Quarto
